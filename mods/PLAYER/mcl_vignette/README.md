# `mcl_vignette`

A server-side playerstep HUD implementation of the Minecraft [Vignette effect](https://minecraft.wiki/w/Screen_effects#Darkness_vignette).
Opacity of the vignette texture is adjusted dynamically based on node light
levels and then is interpolated over a period of time. This effect can be
disabled in player settings under "Graphics".
